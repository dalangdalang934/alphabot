# BSC 交易自动化工具

## 🔖 项目概述

本工具是一个用于在币安智能链（BSC）上执行自动化交易操作的脚本。该工具仅用于**研究与教育目的**，旨在帮助用户了解区块链交易机制和智能合约交互原理。

## 🛠️ 使用说明

1. 安装必要的依赖：
   ```
   pip install -r requirements.txt
   ```

2. 配置.env文件：
   - 设置RPC_URL（BSC节点URL）
   - 设置PRIVATE_KEY（钱包私钥，不带0x前缀）
   - 设置合约地址和其他参数

3. 运行脚本：
   ```
   python tas.py
   ```
